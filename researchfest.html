My Research Festival 2025

Research Prompt: Modern Editorial Blog

Project Specification

"Design a modern editorial-style blog website inspired by a clean and trendy lifestyle blog layout. The UI should focus on visual storytelling, featured posts, and a structured content feed, with large images, bold typography, and minimal clutter."

🧩 Layout Structure

  • 🖼 Hero Section: Featured Post Highlight with full-width banner, category labels, and bold CTA.
  • 📰 Latest Posts: 2–3 column grid of clean, equal-sized blog cards.
  • 📌 Categories: Horizontal filter tabs for seamless navigation.
  • 📷 Highlight Section: "Stories that inspire" editorial feature.
  • 🧑 Author Section: Personal branding with circular profile and bio.
  • ✉️ Newsletter: Minimal subscription layout for updates.

🎨 Design Style Guide

  • Colors: White background, Dark gray text, Soft beige accents.
  • Typography: Bold serif headings, Clean sans-serif body.
  • UI Effects: Image hover zoom, smooth scroll fade-in, sticky navbar.

🧠 UX Concept

"A modern digital magazine"

Focuses on Content Hierarchy and Visual Storytelling to ensure an effortless reading experience.
